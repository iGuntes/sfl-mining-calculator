Параметры

Оставлен только безопасный источник через SFL World proxy.
Рыночная база использует цены рынка на ингредиенты, Собственная добыча — внутреннюю себестоимость своей цепочки, Гибрид — собственную цепочку для крафта инструмента, но сравнение выгоды идёт с ценой продажи на рынке.
По среднему дропу — стоимость предыдущих звеньев делится на твой средний выход ресурса. Фиксированный рецепт — жёсткая COIN‑стоимость инструмента по рецепту без деления на дроп.
Применяется для p2p. Для seq комиссия не учитывается.
Задаётся от обратного: сколько COIN стоит 1 FLOWER. По умолчанию 1 FLOWER = 1000 COIN.

Подсказка по логике: «Лучший способ получить» — это как дешевле получить ресурс для себя прямо сейчас. «Совет для продажи» — это рекомендация: имеет ли смысл добывать этот ресурс именно для продажи по текущей цене после комиссии.
Логика: «Купить с рынка» показывает прямую цену покупки ресурса на рынке. «Себестоимость» — это стоимость производства 1 ед. ресурса по выбранной модели цепочки. «Роль в цепочке» описывает, как именно ресурс может быть полезен: как сырьё, для собственного использования или для продажи.
