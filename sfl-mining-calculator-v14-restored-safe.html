Подсказка по логике: Лучший способ получить — это как дешевле получить ресурс для себя прямо сейчас. Решение — это выгодно ли добывать этот ресурс именно на продажу по текущей цене после комиссии. Поэтому сочетание «Производить вручную» + «Дешевле купить» означает: для собственного использования ресурс дешевле сделать самому, но для заработка на продаже его добывать невыгодно.
Логика: колонка Купить с рынка показывает прямую цену покупки ресурса на рынке. Колонка Произвести по рынку показывает себестоимость производства с рыночной оценкой ингредиентов, а Произвести вручную — себестоимость через твою собственную цепочку добычи. COIN часть с цепочкой добавляет монетную нагрузку всех предыдущих ресурсов через средний дроп; например для Stone это 3 × (16 / дроп Wood) + 16 COIN.
